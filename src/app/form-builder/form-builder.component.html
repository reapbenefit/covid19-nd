User Instructions while adding buttons in Formbuilder
For Location Button

  • In Display tab ---> Action should be "Event" and Button Event should be "click", Custom CSS Class should be "custom_btn_css"
  • In logic tab ---> Add Logic ---> logic name should be "getforminputdata" and select Trigger --->Type should be "Event" and Event Name should be "click".

    • For Add User Button

  • In Display tab ---> Action should be "Event" and Button Event should be "click", Custom CSS Class should be "add_user".
  • Follow second point of location button instruction.

    • For Add Organisation Button

  • In Display tab ---> Action should be "Event" and Button Event should be "click", Custom CSS Class should be "org_list".
  • Follow second point of location button instruction.

    • For File Upload

  • In File tab ---> Select Storage as "Base64" and Save.
